# Test Data

Sample data for QA testing and demos of Qori's slash commands.

## Available Test Studies

| Folder | Description |
|--------|-------------|
| `va-health-benefits-mobile-app/` | VA mobile app navigation study with 3 participants |

## How to Use

Each study folder is organized by slash command. Open the folder for the command you're testing, copy the sample content, and paste into the Qori modal.

## Adding New Test Studies

Create a new folder following this structure:

```
test/[study-name]/
├── README.md
├── qori-request/
├── qori-plan/
├── qori-participants/
├── qori-outreach/
├── qori-notes/
├── qori-analyze/
├── qori-synthesis/
└── qori-report/
```
