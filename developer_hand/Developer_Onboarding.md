# 👋 Welcome to CivicMind Dev

Thanks for helping build the MVP. This is an AI tool for civic researchers and designers to automate their research workflows.

## 🔧 What You'll Work On

- Slackbot commands: `/civicmind ask`
- Transcription and tagging: `interview_01.wav`
- Summary generation → Markdown reports
- GitHub/Jira integration (create Issues from insights)

## 🛠️ Environments to Know

- `/data/` contains transcripts and research files
- `/slackbot/` is the Slackbot server
- `/reporting/` builds summaries
- `/docs/` contains onboarding + ethical AI principles

## 🚨 Security + Ethics

- No AI output is final — human review is required
- Keep PII out of logs and commits
- See `docs/ai-ethics.md` for guidance
